How Slots Work

Named Slots

The <elsa-callout> component defines three slots:

  • slot="icon" - Optional icon (any HTML, not just emoji!)
  • slot="title" - Optional heading
  • Default slot (no name) - Body content

You provide the content, the component provides the structure and styling!

How Component Communication Works

Smart Connections

  1. Control finds Snowfall: Uses target attribute as CSS selector
  2. Slider moves: Triggers input event
  3. Control updates Snowfall: Calls snowfall.setIntensity()
  4. Snowfall adjusts: Creates/removes snowflakes and updates speeds
  5. Events emitted: Both components emit custom events for other listeners
Usage
<elsa-snowfall density="50" speed="5"></elsa-snowfall>

<elsa-snow-control target="elsa-snowfall" initial-value="50">
  Snow Intensity
</elsa-snow-control>
Event Listening

Custom Events

Both components emit events you can listen to:

// Listen to control changes
control.addEventListener('elsa-snow-change', (e) => {
  console.log('Intensity:', e.detail.intensity);
});

// Listen to snowfall changes
snowfall.addEventListener('elsa-snowfall-change', (e) => {
  console.log('Snowflakes:', e.detail.count);
});

Open your console and move the slider above to see events!
