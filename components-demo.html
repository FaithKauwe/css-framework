What is the Elsa Framework?

Elsa is a lightweight, classless CSS framework with an Ice Queen aesthetic. It provides beautiful defaults for semantic HTML and a robust component system built with BEM naming and design tokens.

How do I use components?

Components use BEM naming: a base class (like .btn), optional element classes (.card__header), and modifier classes (.btn--primary).

Example: <button class="btn btn--primary">Click Me</button>

What are design tokens?

Design tokens are CSS custom properties that define all design decisions: colors, spacing, typography, etc. They make the framework consistent and easy to customize.

All Elsa components use tokens like var(--color-ice-3) instead of hard-coded values.

Can I customize the framework?

Yes! Override any design token in your own CSS to customize colors, spacing, fonts, and more. The modular file structure makes it easy to extend.
